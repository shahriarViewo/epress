RESPONSIVE FONT SIZE PATTERNS:
==============================

Mobile First Approach:
- Base sizes typically start at mobile (text-sm, text-base)
- Increase on tablet (md:text-base, md:text-lg)
- Further increase on desktop (lg:text-lg, lg:text-xl)
- XL screens get largest sizes (xl:text-xl, xl:text-2xl)

Common Responsive Patterns:
- text-sm md:text-base lg:text-lg (14px → 16px → 18px)
- text-base lg:text-lg (16px → 18px)
- text-3xl md:text-4xl (30px → 36px)
- text-2xl md:text-3xl (24px → 30px)

FONT WEIGHTS:
=============
- font-normal - Regular weight (400)
- font-medium - Medium weight (500)
- font-semibold - Semi-bold (600)
- font-bold - Bold (700)

TYPOGRAPHY HIERARCHY:
====================
1. Headlines: text-4xl to text-5xl font-bold
2. Section titles: text-2xl to text-3xl font-bold
3. Card titles: text-lg to text-xl font-semibold
4. Body text: text-base to text-lg
5. Small text: text-sm
6. Labels/Metadata: text-xs

NOTES:
======
- Project uses Tailwind CSS for all styling
- Consistent use of gray scale for text hierarchy
- Brand orange (#F24E1E family) used for CTAs and accents
- Responsive design with mobile-first approach
- Font sizes scale progressively across breakpoints
- Color system uses semantic naming (primary, secondary, muted)
